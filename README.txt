Привет!Это - краткое руководство по использованию bookTopia.

Для начала, ты попадаешь на главную страницу.
 1.Если ты в первый раз на нашем сайте то ты можешь зарегистрироваться и пройти входной опрос на предпочтения по профессиям чтобы мы понимали что тебе стоит рекомендовать.
  2.Если ты уже зарегистрировался на сайте, то тебе будут предложены профессии по твоим предпочтениям или по которым ты уже проходил тест.
   3.Если не собираешься регистрироваться то на главной странице тебе будут предложены самые просматриваемые профессии.
Содержание главной страницы: карточки с профессииями, поиск по ключевым словам, кнопка для авторизации/регистрации.

Если нажмешь на кнопку перейти к профессии то увидишь список наыков требующихся для этой профессии(они кликабельны и если ты на них нажмешь то увидишь краткую информацию о навыке, какие поднавыки он содержит и сможешь пройти тест по этому навыку), а также её краткое описание и кнопку для прохождения общего теста по данной профессии.

На странице с тестом ты должен выбирать правильные варианты ответа на заданные вопросы путем клика на нужный вариант.

По итогу теста ты увидишь свой результат, сможешь посмотреть ответы, а твои результаты будут сохранены в профиле(если ты зарегистрировался)
